# GHOST PACK

Start a new kali, and then simple as this;

1. Update

        apt-get update && apt-get upgrade -y
        
2. Clone

        git clone https://github.com/NeverWonderLand/ghostpack

3. Run

        cd ghostpack
    
        chmod +x ghostpack.sh
    
        ./ghostpack.sh

4. Go take a walk (around 3-3.5 hours)
